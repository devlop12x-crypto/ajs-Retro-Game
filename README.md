# 🎮 Retro Game

Браузерная пошаговая RPG-игра на чистом JavaScript. Игрок управляет командой героев на клетчатом поле 8×8, сражается с командой противника, проходит уровни и может сохранять прогресс.

## Игровой процесс

- Выбери персонажа кликом — под ним появится жёлтый круг выделения
- Наведи на пустую клетку — зелёная подсветка означает, что туда можно переместиться
- Наведи на врага — красная подсветка означает, что можно атаковать
- После хода игрока компьютер автоматически делает свой ход
- Победи всех врагов на уровне — команда получит повышение уровня и начнётся следующий
- Проигрыш — если все твои персонажи погибают

## Персонажи

| Сторона | Классы |
|---------|--------|
| Игрок | Bowman, Swordsman, Magician |
| Противник | Vampire, Undead, Daemon |

Каждый персонаж имеет характеристики: `level`, `attack`, `defence`, `health`, а также индивидуальные дальности хода и атаки.

При наведении на персонажа отображается tooltip:
```
🎖1  ⚔25  🛡15  ❤50
```

## Архитектура

```
src/
├── characters/
│   ├── Bowman.js
│   ├── Swordsman.js
│   ├── Magician.js
│   ├── Vampire.js
│   ├── Undead.js
│   └── Daemon.js
├── Character.js          # Базовый класс персонажа
├── GameController.js     # Главный контроллер игры
├── GameState.js          # Состояние игры + десериализация
├── GameStateService.js   # Сохранение/загрузка через localStorage
├── generators.js         # Генерация и расстановка команд
├── utils.js              # canMove, canAttack, calcTileType и др.
├── themes.js             # Темы уровней (prairie, desert, arctic, mountain)
└── ...
```

**Ключевые классы:**

- `Character` — абстрактный базовый класс, запрещает прямое инстанцирование через `new.target`; реализует `levelUp()` с формулой роста характеристик
- `GameController` — управляет всеми событиями доски, логикой ходов, атак, перемещений и смены уровней
- `GameState` — хранит текущее состояние; статический метод `from()` восстанавливает объект из plain-данных с реконструкцией экземпляров классов
- `GameStateService` — сохраняет и загружает состояние через `localStorage` с обработкой ошибок
- `generators.js` — генератор персонажей (`function*`) и расстановка команды на разрешённые колонки без коллизий

**Логика хода компьютера** — перебирает всех врагов и всех игроков, выбирает пару с минимальной дистанцией, атакует если достаёт, иначе двигается в сторону цели.

**Формула урона:**
```js
damage = Math.max(attack - defence, attack * 0.1)
```

## Уровни и темы

| Уровень | Тема |
|---------|------|
| 1 | Prairie (луга) |
| 2 | Desert (пустыня) |
| 3 | Arctic (арктика) |
| 4 | Mountain (горы) |

С каждым уровнем команда врагов становится больше и сильнее. После 4-го уровня — победа.

## Установка и запуск

```bash
git clone https://github.com/devlop12x-crypto/ajs-Retro-Game.git
cd ajs-Retro-Game
npm install
npm start
```

Откроется на `http://localhost:8080`

### Сборка для продакшена

```bash
npm run build
```

Результат — в папке `dist/`

### Тесты

```bash
npm test          # запуск тестов
npm run coverage  # покрытие кода
```

78 тестов, покрывающих: генераторы, логику движения и атаки, уровни персонажей, сохранение/загрузку состояния.
